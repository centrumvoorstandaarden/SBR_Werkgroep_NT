# NTA 2.02.02.12

 2.02.02.12 | . 
 :--- | :--- 
 Naam | VERVALLEN @form MOET NIET op <xs:element> geplaatst worden 
 Toelichting | Met het @form attribuut kan de @elementFormDefault uit <xs:schema> overruled worden. Met de waarde ‘qualified’ wordt afgedwongen dat aan dit element alleen gerefereerd kan worden met een QName (Qualified Name) dus met een namespace prefix identifier voor de @name. Dit is de common practice bij XML schema’s.
FRTA 4.2.4 stelt dat <xs:schema elementFormDefault=”qualified”/>, dus is het enige logische gebruik dat de waarde ‘unqualified’ op elementniveau gevoerd kan worden.  Dit wordt door FRTA verboden.
